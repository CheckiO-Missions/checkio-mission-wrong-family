Michael was always knew that there are something wrong with his family. So he found a very old and pretty heavy book that listen all the relationships in his family. Help Mike to figure out is there any strange things in the book.

You have list of family relationships between father and son. Every element of this list has two elements. First is father's name, second is son’s name. All names in Mike’s family are unique.

Input: list of list. Every element has two strings. List has at least one element

Output: bool. True - if all the relationships has their own place on the family tree where only one person doesn't have a father listened in this book.
